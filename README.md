## **`Caracteristicas:`**

- Funcion para `Abrir` documentos de texto .txt
- Funcion para `Guardar` y `Guardar como` el documento
- Funcion de `Salir` 
- Muestra el numero de columna y línea 
- Muestra mensaje de guadar archivo cuando se cierra desde el menu de `Abrir`
